O básico do HTML

Neste primeiro tópico, falaremos o básico do HTML.

Documentos HTML

Ao começar um arquivo, deve-se falar qual é o tipo de documento utilizado como: "<!DOCTYPE html>".

Após identificado, todo documento começa com "<html>" e termina com"</html>".

Tudo que é visível dentro do site é feito por outras tags chamadas: "<body>" e "</body>".

EXEMPLO

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

RESULTADO

My First Heading

My first Heading

A tag <!declaração> DOCTYPE

Indica qual é o tipo de documento e serve de referência para poder exibir as páginas da Web corretamente: "<!DOCTYPE>".

Aparece somente uma vez, no topo do código junto com as outras informações base.

A declaração não diferencia letras maiúsculas nem minúsculas.

É a mesma para identificar HTML5.

EXEMPLO

<!DOCTYPE html>

Títulos HTML

Os títulos são definidos por 6 tipos desde o "<h1>" até o "<h6>".

O primeiro é usado para títulos mais importantes e o restante para subtítulos e textos.

EXEMPLO

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h1>This is heading 4</h4>
<h2>This is heading 5</h5>
<h3>This is heading 6</h6>

RESULTADO

This is heading 1

This is heading 2

This is heading 3

This is heading 4

This is heading 5
This is heading 6

Parágrafos HTML

Todo parágrafo é feito pela tag: "<p>

EXEMPLO

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

RESULTADO

This is a paragraph.

This is another paragraph.

Links HTML

Os links são definidos pela tag "<a>" e destino feito pelo atributo "href".

Os atributos são informações adicionais sobre os elementos do HTML.

EXEMPLO

<a href="https://www.youtube.com">This is a link</a>

Imagens HTML

As imagens são inseridas com a tag "<img>".

Assim como os links, as imagens precisam de atributos.

Os atributos para essa tag são o arquivo de origem (src),
o texto alternativo (alt), e as dimensões (width e height).

EXEMPLO

<img src="youtube.jpg" alt="youtube.com" width="104" height="142">
